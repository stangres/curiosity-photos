# Mars Curiosity Rover Photos

The app shows the latest images taken by the Curiosity rover.

## Install

### `npm i`

## Run

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
